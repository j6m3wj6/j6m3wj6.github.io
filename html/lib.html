cb_axis_handleTextChanged(event, canvas_id, ax_id)[source]¶

This function connect with all QComboBox components of the form in the dialog. It will be triggered when any comboBox’s option is changed.

Actually, it is not allowed to have same options on different comboBoxes except ‘None’ option. Every two options would automatically switch their positions if one of those is changed.

Parameters

  • event (str) – Current option text which is a CurveType name

  • canvas_id (int) – The index of the canvas that need to modify its axis’ CurveType

  • ax_id (int) – The axis index that need to modify its CurveType. 0 is main axis, 1 is sub axis.
